    Producciones

    Puedes crear producciones y seguir todo el porcesos de estas hasta que esten finalizadas ademas de llevar el control del costo de produccio y otros.

    Diseños

    Crea tus disños en una imagen y guardalos para tener un mejor control de tus productos y diseños de la diferentes producciones que realizaras.

    Productos

    Puesdes crear cualquier tipo de producto para luego realizar diseños sobre este y crear producciones.

    Calculos

    Antes de crear tus producciones podras hacer calculos para estimar cuanto material te llevara o cuantas unidades puedes hacer con el material disponible en inventario.

    Grupos

    Crea tus grupos de trabajo con tus compañeros para poder realizar buenas producciones.
